Users
============

Welcome
-------
Jazzee is a colaborative project aimed at creating a very flexible application system.  Our goal
is to build a system which requires no developer intervention when creating new applications.

Requirements
------------

Jazzee requires a minimum of PHP 5.3.0. For greatly improved
performance it is also recommended that you use APC with PHP.
